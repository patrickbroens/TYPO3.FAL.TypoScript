**************************************
Working with Content Objects (cObject)
**************************************

When rendering FAL objects, there are three content objects which are important. IMAGE/IMG\_RESOURCE are meant for
rendering a single image. For multiple images, or other FAL objects, the new content object FILES has been introduced.
